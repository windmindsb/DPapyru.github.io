# 测试文档

这是一个测试文档，用于验证 GitHub Pages 是否能够正确访问位于 docs 目录下的 Markdown 文件。

如果你正常情况下看到了这个文档，一般是两种情况:

- 此页面已丢失/链接失效
- 文章正在构建

## 测试目的

验证 GitHub Pages 对以下路径的访问能力：

- 直接位于 docs 目录下的 Markdown 文件
- 二级子目录中的 Markdown 文件

## 测试内容

如果您能够看到这个页面，说明 GitHub Pages 可以成功访问位于 docs 目录下的 Markdown 文件。

### 代码示例

```csharp
// 这是一个 C# 代码示例
public class TestClass
{
    public void TestMethod()
    {
        Console.WriteLine("Hello, GitHub Pages!");
    }
}
```

## 结论

如果此文档能够正常加载和显示，说明路径处理逻辑正确，GitHub Pages 可以访问 docs 目录下的文件。
